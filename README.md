# zshort

A custom link shortener

## Running

Create a `.env` file with your MongoDB connection information, either as a URI in `MONGO_URI` or individual parts:
- `MONGO_HOST`
- `MONGO_USER`
- `MONGO_PASS`
- `MONGO_PORT`

Then, run `uvicorn zshort:app --port=<port> --workers=<workers>` to run the app
